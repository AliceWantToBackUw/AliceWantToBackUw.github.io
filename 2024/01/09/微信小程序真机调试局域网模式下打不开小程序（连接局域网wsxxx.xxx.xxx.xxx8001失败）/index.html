# 报错

# 解决方式

确保手机和电脑连接同一个局域网；不要勾选 局域网模式

# 完整步骤

  • 查看本机的 IP 地址
    Win + R 输入 cmd 打开命令行工具，使用 ipconfig 查看本地服务器 ip
  • 配置小程序的请求路径
    不要使用 localhost ，改成查询到的 IP地址 和 端口号

  • 关闭防火墙
  • 勾选 “微信开发者工具” 里的 不校验合法域名…… 选项

    调式窗口里的 不校验合法域名…… 可勾可不勾
